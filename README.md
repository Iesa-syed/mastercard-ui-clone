# 💳 Credit Card UI Design

A modern Mastercard-inspired credit card component built using pure HTML and CSS.




## 🚀 Features

- Clean credit card design
- Flexbox layout
- Custom background styling
- Responsive card container
- Mastercard branding elements
- Beginner-friendly code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

---
